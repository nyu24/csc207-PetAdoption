# Pet Adoption Simulator

This is a simple game that revolve around taking care of a pet of the user's choosing
from the PetFinder API. The goal of the game is to show that the user is capable of
taking care of before the vet arrives for their checkup. If the user is shown to be
an affectionate and proficient pet caretaker, they will be directed and suggested
to adopt the pet they virtually raised!

User Stories:
* As a user, I want to be able to look through a list of pets (with info and pictures) to 
choose a pet to take care of, and I want to be able to look at pets based on species. Ex. dogs.
* As a user, I want to be able to click various buttons with various functionalities to simulate
taking care of a pet.
* As a user, I want to be able to view and see information regarding a pet’s living space to know 
if my space can properly house a pet.
* As a user, I want to be able to click a link (redirect) to see my chosen animal’s petfinder page
* As a user, I want to be able to save/load my game by clicking buttons so that I can continue it 
later on.
* As a user I want there to be conditions to win or lose the game to know how well I can take care 
of a pet.
* As a user, I want to see my high scores, as well as how my current score compares to these.

API's Usage:
PetFinder: This API is used for displaying pets near the user that is up for adoption.

PetFinder API: https://www.petfinder.com/developers/
PetFinder Doc: https://www.petfinder.com/developers/v2/docs/
